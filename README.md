# Convert a directory into a printable document

## Install

```shell
uv tool install git+https://github.com/josephabbey/dir-to-docx.git
```

## Usage

```plaintext
Usage: dir-to-docx [OPTIONS] INPUT_DIR OUTPUT_FILE PATH_PREFIX                                  
       
Options:
  --exclude TEXT     many exclusions, e.g.: `*.ico`
  --associate TUPLE  many associations, e.g.: `*.svg,xml`
  --help             Show this message and exit.
```
